All samples acquired from the sensor, must be placed inside the 'Samples' folder identified by a unique sequencial number.

In order to use this code, you must have your samples at Implementação/Samples/

To make all samples available to the code, run the script 'getInputImages.m'.
To generate all the texture descriptors, run the script 'generateTextureDescriptors.m'
To evaluate an image, run the script 'main.m'
